# NAME

App::mii - Internals for mii

# SYNOPSIS

```
$ mii help
```

# DESCRIPTION

App::mii is just for me.

If I forget how to use mii, I could install it and run `mii help` or I could check the POD at the end of
`script/mii.pl`
